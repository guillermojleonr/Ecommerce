- Install django
pip install django==4.1

- Check installation and django version
(interactive console)
import django
django.VERSION

-SSL in development environment
pip install django-sslserver
include sslserver in installed apps in settings.py
run python manage.py runsslserver 0.0.0.0:8888

- Install and configure django-environ
pip install django-environ
https://alicecampkin.medium.com/how-to-set-up-environment-variables-in-django-f3c4db78c55f
Secure SECRET_KEY, DB env variables and other sensitive data

- Create new project
cd rootDir
django-admin startproject projectname

- Set up .gitignore
There is a boilerplate in ml_querier project, include .env, *.ipynb

- Create new App
cd projectDir
python manage.py startapp appName
Register new app in settings.py
Check if app is registered ok
python manage.py check appName
python manage.py runserver
Create or migrate views
Create or migrate html templates
Register url's in app's urls.py and update project's urls.py

- Admin panel
- Create superuser
python manage.py createsuperuser

- Create database sqlite3 based on models.py
Create model classes in models.py
cd projectDir
python manage.py makemigrations
python manage.py sqlmigrate appName migrationNumber
python manage.py migrate

- Register in admin panel the new app and its model classes to be able to use de admin panel to interact with the tables
